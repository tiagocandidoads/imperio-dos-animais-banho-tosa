# 🐾 Sistema de Banho e Tosa - Império dos Animais

Sistema web centralizado para monitoramento de banho e tosa de 4 lojas com dashboard e relatórios mensais.

## 📋 Funcionalidades

- ✅ Login por loja
- ✅ Registro diário de banhos e tosas
- ✅ Dashboard centralizado (admin)
- ✅ Gráficos de quantidade por loja
- ✅ Relatórios mensais
- ✅ Histórico completo

## 🛠️ Stack Tecnológico

- **Frontend**: React + TypeScript
- **Backend**: Node.js + Express
- **Banco de Dados**: PostgreSQL
- **Autenticação**: JWT
- **Gráficos**: Recharts
